# AndyAI News v1.4.0-b — TTS Provider Adapter Layer Pack

This pack introduces the provider abstraction layer for audio rendering.

Included:
- canonical TTS provider adapter types
- provider config helpers
- admin provider inspection page
- API route for provider snapshot
- provider snapshot data

This is a minimal safe overlay and does not yet include:
- real vendor SDK integration
- public audio player
- asset storage/publishing
- archive audio lane
