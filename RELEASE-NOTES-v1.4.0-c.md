# AndyAI News — v1.4.0-c

## Audio Asset Storage & Metadata Pack

This release introduces the canonical storage and metadata layer for audio assets.

### Added
- `lib/audio-storage/*`
- `/admin/audio/assets`
- `/api/audio/assets`
- `data/audio-storage/assets-snapshot.json`

### Purpose
- separate internal storage from public projection
- standardize audio asset metadata
- prepare transcript/captions/public URL linkage
