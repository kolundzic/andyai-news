# Admin Shell v1.0

## Route

```bash
http://localhost:3000/admin
```

## Purpose

This is the stable MVP operator panel for daily AndyAI News publishing.

## Core blocks

- import existing JSON
- create new day draft
- form editor + raw JSON mode
- validation panel
- cover image helper
- publish checklist
- local publish pack helper

## v1.0 note

This version intentionally stays focused on a simple local-first publishing flow.
It does not try to become a full CMS yet.
