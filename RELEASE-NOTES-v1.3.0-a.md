# AndyAI News — v1.3.0-a — Publish State & Release Discipline

This pack introduces the first delivery-grade publishing state layer for AndyAI News.

## Included
- canonical publish state types
- publish/release config
- helper functions for status, transitions, and gating
- `/admin/publish` overview page
- `/api/admin/publish` JSON endpoint
- sample publish snapshot data

## Intent
This is a minimal safe overlay that starts the `v1.3.0` publishing/delivery series without disturbing `/admin/v110a`.
