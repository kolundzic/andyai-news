# andyai.news — Editorial Policy Pack

**Status:** locked  
**Version:** v0.1  
**Scope:** pilot  
**Owner:** Andy + AI  
**Purpose:** Defines editorial standards, tone, trust rules, and anti-spam discipline for the pilot.

━━━━━━━━━━━━━━━━━━━━

## Editorial north star

**Accurate + clear + useful + readable + calm**

━━━━━━━━━━━━━━━━━━━━

## Core editorial rules

- source-grounded writing only
- no hallucination tolerance
- facts first, interpretation second
- readable professional language
- no fluff / no filler
- original synthesis, not copy-paste
- context is mandatory
- uncertainty must be visible
- sensitive-topic caution
- no fake urgency / no clickbait inflation
- editorial consistency across the site
- update discipline when facts change

━━━━━━━━━━━━━━━━━━━━

## Tone of voice

The site should sound:

- professional
- calm
- clear
- useful
- modern
- non-tabloid
- non-robotic
- non-hype

━━━━━━━━━━━━━━━━━━━━

## Canonical statement

**andyai.news must not sound like AI producing text.  
It must sound like a serious digital magazine using AI in a disciplined way.**
