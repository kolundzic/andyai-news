# andyai.news — Source Usage Micro-Routine

**Status:** locked  
**Version:** v0.1  
**Scope:** pilot  
**Owner:** Andy + AI  
**Purpose:** Defines the pre-draft micro-routine for validating sources before a topic enters draft mode.

━━━━━━━━━━━━━━━━━━━━

## Micro-routine

1. identify the signal
2. find the original source
3. classify source type
4. assign trust level
5. isolate the fact anchor
6. check topic fit
7. test novelty / usefulness
8. check redundancy
9. cross-check if needed
10. apply sensitivity mode if needed
11. pass hype filter
12. decide: draft / park / reject

━━━━━━━━━━━━━━━━━━━━

## Canonical rule

**Draft is a privilege, not an automatic right of every signal.**
