# andyai.news — SEO Hygiene Checklist

**Status:** active  
**Version:** v0.1  
**Scope:** pilot  
**Owner:** Andy + AI  
**Purpose:** Defines healthy SEO discipline based on clarity, clean structure, and non-spammy publishing logic.

━━━━━━━━━━━━━━━━━━━━

## SEO philosophy

SEO for andyai.news is not about gaming algorithms.

It is about:

- clear titles
- clean slugs
- sensible meta descriptions
- stable category logic
- useful archive pages
- internal linking that helps
- avoiding thin content
- mobile readability
- clean indexing structure

━━━━━━━━━━━━━━━━━━━━

## Minimum pass per article

- clear title
- sensible slug
- meta description
- one primary category
- clean heading structure
- basic related/internal link logic
- archive placement
- no thin-content feel

━━━━━━━━━━━━━━━━━━━━

## Canonical rule

**SEO is order and clarity, not manipulation.**
