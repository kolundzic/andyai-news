## Pipeline

**signal → verify → fit → angle → notes → outline → draft → review → revise → package → publish → archive → check → update**

━━━━━━━━━━━━━━━━━━━━

## Main stages

1. Signal Intake
2. Source Verification
3. Topic Fit Check
4. Angle Selection
5. Research & Notes
6. Article Structure
7. Draft Writing
8. Editorial Review
9. Revision Pass
10. Metadata & Packaging
11. Publish
12. Archive Placement
13. Post-Publish Check
14. Update / Correction Loop

━━━━━━━━━━━━━━━━━━━━

## Canonical rule

**If the article fails source gate or editorial gate, it does not go out.**
