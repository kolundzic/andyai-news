# AndyAI News v1.0 — First Production Polish Pack

## Locked scope

This v1.0 release locks the MVP around a simple, stable workflow:

1. Create or import a daily dataset in `/admin`
2. Validate the dataset
3. Fix the cover image path
4. Review the publish checklist
5. Review the local publish pack helper
6. Export `news-YYYY-MM-DD.json`
7. Place the cover image in `public/covers/YYYY-MM-DD-cover.jpg`
8. Update `data/manifest.json` when the new day becomes live

## What was polished

- cleaner admin layout
- production lock panel
- tighter visual rhythm in admin
- clearer stable-MVP positioning
- no scope expansion beyond the daily publishing workflow

## MVP status

Stable enough for first real daily use.
