# AndyAI News — v1.3.0-c

## Locale-specific Public Editions

This pack introduces locale-aware public edition routing and a small deterministic public edition contract.

### Added
- `lib/public-editions/types.ts`
- `lib/public-editions/config.ts`
- `lib/public-editions/helpers.ts`
- `app/[locale]/edition/[day]/page.tsx`
- `app/api/editions/public/route.ts`
- `data/public-editions/edition-snapshot.json`

### Purpose
Move AndyAI News from internal publishing discipline toward locale-specific public editions with day-aware public pages.
