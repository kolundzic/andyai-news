# AndyAI News — Public Surface CSS-First Polish v0.3

## Release type
CSS-first editorial polish pack

## Status
Completed

## Goal
Improve visual hierarchy, readability, spacing rhythm, and editorial feel using the existing custom CSS system only.

## Included
- stronger hero/header presentation
- improved card spacing and hover feel
- cleaner pill/meta treatment
- better story detail hierarchy
- refined progress/navigation surfaces
- mobile-safe spacing and rhythm polish

## Why this matters
After the recovery from the Tailwind mismatch, this pack intentionally keeps the project inside its native CSS-first styling model while still improving the public reading experience.

## Scope discipline
No Tailwind dependency.
No admin expansion.
No routing rewrite.
No backend/data model changes.
