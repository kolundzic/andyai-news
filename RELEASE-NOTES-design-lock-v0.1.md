# AndyAI News — Design Lock Pack v0.1

## Release type
Design governance / safety pack

## Status
Completed

## Goal
Protect the stable public visual baseline and prevent future design regressions during polish work.

## Included
- stable baseline commit recorded
- design-sensitive public files identified
- CSS-first rule formalized
- small-step visual workflow defined
- rollback/recovery rule documented

## Why this matters
Recent iterations showed that the project can remain technically build-clean while still losing its intended public visual identity.

This pack formalizes guardrails so future polish work stays incremental, reversible, and visually safe.
