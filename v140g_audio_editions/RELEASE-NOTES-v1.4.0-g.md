# AndyAI News — v1.4.0-g
## Locale-Aware Audio Editions Pack

This release introduces locale-aware audio edition modeling for AndyAI News.

Included:
- canonical audio edition types
- locale-specific readiness/fallback config
- helper layer for locale-aware audio edition binding
- `/admin/audio/editions`
- `/api/audio/editions`
- `data/audio-editions/editions-snapshot.json`

This is a minimal safe overlay and does not yet introduce archive/replay audio lane or full public player automation.
